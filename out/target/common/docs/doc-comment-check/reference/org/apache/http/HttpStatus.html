public interface

HttpStatus

org.apache.http.HttpStatus

Class Overview

Constants enumerating the HTTP status codes. All status codes defined in RFC1945 (HTTP/1.0), RFC2616 (HTTP/1.1), and RFC2518 (WebDAV) are listed.
