URLClassLoader
URLClassLoader This class loader is responsible for loading classes and resources from a list of URLs which can refer to either directories or JAR files. 

Class Overview

SecureClassLoader represents a ClassLoader which associates the classes it loads with a code source and provide mechanisms to allow the relevant permissions to be retrieved.
